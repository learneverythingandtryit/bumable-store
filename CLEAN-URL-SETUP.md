1. Create GitHub organization: `bumable-store`
2. Transfer this repository to the organization
3. Enable GitHub Pages
4. Result: `https://bumable-store.github.io/`

### Option 2: Username Repository
1. Create new GitHub account: `bumable-store`
2. Create repository: `bumable-store.github.io`
3. Push your code there
4. Result: `https://bumable-store.github.io/`

### Option 3: Custom Domain (Costs ~$10/year)
